# 🌿 Dew Routes

**Explore Wayanad, Your Way**

Dew Routes is a modern, responsive tourism landing page focused on showcasing Wayanad travel experiences. The website is designed to help travelers discover trekking adventures, waterfalls, sightseeing destinations, and customized travel packages while enabling direct WhatsApp enquiries.

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Adventure-inspired UI with modern styling
- Featured Wayanad experiences
- Image gallery showcasing scenic locations
- Direct WhatsApp enquiry integration
- Smooth scrolling navigation
- Clean and lightweight frontend

## 🏕️ Experiences

- Trek & Trails
- Waterfall Explorer
- Custom Wayanad Journey

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 📱 WhatsApp Integration

Visitors can directly connect through WhatsApp for:
- Trip planning
- Package enquiries
- Customized itineraries
- Travel assistance
